Prueba Técnica de Inteligencia Artificial - SUMMA

Este repositorio contiene la solución completa a la Prueba Técnica de IA, que aborda varios desafíos de Machine Learning, Deep Learning y Agentes de IA.

1. Agente/ --> Contiene la arquitectura y el código de simulación para el Agente Orquestador de RRHH , que decide entre usar su conocimiento base o llamar a funciones.

2. API_Clasificacion_Alpha_Betha/ --> Implementación de una API web (posiblemente FastAPI o Flask) para desplegar un modelo de clasificación (ej. binaria o multiclase).

3. Pronostico serie de tiempo/ --> Modelos y notebooks para la predicción de valores futuros (Forecasting), probablemente utilizando ARIMA, Prophet o modelos LSTM.

4. Red neuronal 2ndo punto/ --> una Solución alternativa al segundo punto que requiere un modelo de Deep Learning una RNA para clasificación (no fue la solución implementada).
